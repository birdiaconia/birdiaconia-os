확인되지 않은 내용 처리 원칙:

- 확인되지 않은 내용을 사실처럼 쓰지 않는다.
- 법률·조례·지원사업은 “최신 확인 필요” 표시를 기본값으로 둔다.
- 출처가 2차 자료뿐이면 “원문 확인 필요”로 표시한다.
- 출처가 충돌하면 충돌 내용을 함께 적고 확정하지 않는다.
- 실행 판단에 영향을 주는 미확인 정보는 후속 확인 과제로 분리한다.

작업 원칙:
- 한국어로 작성한다.
- 원본 출처 없이 단정하지 않는다.
- 특정 내용이 확인되지 않으면 “추가 확인 필요”라고 표시한다.

- 해외 사례의 장점뿐 아니라 한계, 비판, 국내 적용 위험도 함께 기록한다.

## 다음 조사자가 먼저 확인해야 할 5가지

1. 이 모델의 공식 원문 출처와 현재 운영 여부.
2. 국내 적용 시 가장 먼저 걸리는 법률·인허가 쟁점.
3. 실행 후보 지역의 최신 조례와 담당 부서.
4. 사용할 수 있는 최신 지원사업 또는 재원 후보.
5. 버디아코니아가 직접 실행할 수 있는 최소 파일럿 단위.
